Запуск на любом компьютере

  1. git clone репозиторий
  2. pip install -r requirements.txt
  3. Скопировать config.example.yaml → config.yaml
  4. Положить свой data/projects.csv (каталог, read-only)
  5. python run_discovery.py scan
  6. UI: streamlit run discovery_app.py → localhost:8501

Каталог используется только для фильтра «уже есть» — на сайт не выкладывается.
